## Overview

This project focuses on visualizing patterns in the Iris dataset using data visualization techniques in Google Colab. The goal is to transform raw data into meaningful visual insights that support analysis and understanding.

---

## Objective

* Visualize patterns in the dataset
* Analyze relationships between features
* Compare multiple attributes effectively

---

## Dataset

**Iris Dataset**

The dataset contains measurements of iris flowers:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

---

## Visualization Tasks

### 1. Bar Chart

* Used to represent categorical distribution of species
* Helps compare the frequency of each class

### 2. Histogram

* Displays distribution of numerical features
* Useful for understanding spread and frequency

### 3. Scatter Plot

* Shows relationships between two features
* Helps identify patterns and clustering among species

### 4. Feature Comparison

* Compared multiple features to observe correlations
* Helps understand variation across different classes
